# AI Task 1 - Image Recognition

## Objective
Train a simple image recognition model using Google Teachable Machine.

## Steps
1. Opened Google Teachable Machine.
2. Created two classes: Dog and Cat.
3. Added three images for each class.
4. Trained the image classification model.
5. Exported the model in TensorFlow (Keras) format.
6. Opened Google Colab and ran the Python code.
7. Uploaded the exported model files.
8. Tested the model using an input image.
9. Saved the prediction output.

## Files
- predict.py
- keras_model.h5
- labels.txt
- output.png
